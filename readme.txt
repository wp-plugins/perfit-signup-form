Inserta un formulario de suscripción en tu página

== Description ==

Esta aplicación permite a los usuarios de tu sitio suscribirse a tus listas de mails de Perfit Email Marketing.

Es posible personalizar los campos y textos para adaptar el formulario a tus necesidades.

== Installation ==

1. Descarga el archivo comprimido que contiene el plugin.
2. Descomprime el contenido del archivo comprimido dentro de `/wp-content/plugins/perfit` y eso es todo!. Ya lo deberías poder ver tu sidebar
